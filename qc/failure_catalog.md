# Failure Catalog v1.2

## 주요 실패 유형
1. Support-surface scale mismatch
   - 의도한 지지 가구 스케일보다 과도하게 작거나 큼
2. Duplicate focal object
   - 장면의 초점 오브젝트가 2개 이상으로 분산됨
3. Clutter in anchor zone
   - keep-out/clearance가 깨지고 주변 오브젝트가 과밀
4. Camera intent drift
   - pitch/yaw/거리 의도에서 벗어남
5. Artifact
   - watermark/signature/text overlay/gibberish text

## 기록 규칙
- 실패 원인은 한 줄 요약
- 다음 시도에서 바꿀 변수는 1개만 지정
